## Usage Instructions

### 1. Manual Workflow Trigger

1. Navigate to the **Actions** tab in your GitHub repository
2. Select **"Financial Intelligence Engine Integration"** from the workflow list
3. Click **"Run workflow"** button
4. Configure the following parameters:

#### Parameters

| Parameter | Description | Default | Options |
|-----------|-------------|---------|---------|
| `phase_selection` | Select phases to execute | `all` | Comma-separated list (e.g., `1,2,3`) or `all` |
| `create_issues` | Create GitHub issues for actionable items | `true` | `true` or `false` |
| `enable_verification` | Enable verification protocols | `true` | `true` or `false` |

### 2. Automatic Triggers

The workflow also runs automatically on:
- **Push** to `main` or `copilot/*` branches
- **Pull Request** to `main` branch
